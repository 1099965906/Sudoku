# 数独计算器
1. 使用Qt5.4编写，在VS2013环境下编译通过。
2. 可以生成1~10个解的数独。
3. 可以用于求解拥有非唯一解的数独。
4. 采用深度优先搜索算法，优先求解可能性低的格子。
5. 可以根据当前解求出上一个或下一个解。
